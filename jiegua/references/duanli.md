# 朱子断例备查（《易学启蒙》考变占）

卦单上已有"取断"一行时，以卦单为准，不必查此表。此表供外来卦使用——`scripts/chagua.py` 内置的就是同一张规则表（与算卦技能的脚本同源），正常情况下跑脚本即可，此处仅作人读备查与解释。

| 动爻数 | 取断 |
|---|---|
| 0（静卦） | 占本卦卦辞 |
| 1 | 以本卦变爻爻辞为断 |
| 2 | 占本卦两变爻爻辞，以居上者为主 |
| 3 | 占本卦及之卦卦辞：本卦为贞，之卦为悔 |
| 4 | 占之卦两不变爻爻辞，以居下者为主 |
| 5 | 占之卦不变爻爻辞 |
| 6 | 乾以用九为断，坤以用六为断，余卦占之卦卦辞 |

几个词的意思，讲解时可能用到：

- **为主**：主辞为纲，别的应据之辞为佐。解读围绕主辞展开，佐辞补充映衬，不平分秋色。
- **贞与悔**（三爻变时）：贞是本卦，读作现状、己方、事情的底子；悔是之卦，读作趋势、变方、事情要去的方向。两条卦辞对读，讲"从什么样变成什么样"。
- **用九、用六**：乾坤六爻皆变时的专辞。用九"见群龙无首"，用六"利永贞"，本身就是极变之际如何自处的指引。
